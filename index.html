Peer-to-Peer File Sharing

In peer-to-peer file sharing, a network or swarm of computers work together to distribute a file, piece by piece. Each computer can get the information it needs, directly from other computers, instead of forcing every computer to download from one server. Each computer connected to the swarm is called a peer. When a computer seeds, it already has a full copy of the file, but it uploads parts of it to benefit the swarm, whereas, when a computer leeches, it has a negative effect on the swarm by downloading a full file and then leaving without seeding, not contributing to the swarm.
